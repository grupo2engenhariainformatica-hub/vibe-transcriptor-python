# Vibe Transcriptor Python

## Overview
Vibe Transcriptor is a Python application that transcribes audio input into text using cutting-edge speech recognition technologies. This tool is designed to enhance productivity by converting spoken language into written format accurately and efficiently.

## Features
- **Automatic Speech Recognition**: Utilizes advanced algorithms to transcribe speech in real-time.
- **Multi-language Support**: Supports transcription in various languages, making it suitable for a diverse user base.
- **User-friendly Interface**: Simple interface for easy navigation and usability.
- **Export Options**: Allows users to export transcriptions in various formats (e.g., TXT, DOCX).

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/grupo2engenhariainformatica-hub/vibe-transcriptor-python.git
   cd vibe-transcriptor-python
   ```
2. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
1. Run the application:
   ```bash
   python main.py
   ```
2. Follow the on-screen instructions to start transcribing.
